# hasibaust13.github.io

Personal academic website of **Khan Md Hasib**, PhD Student at UNSW Canberra.

**Live site**: [https://hasibaust13.github.io](https://hasibaust13.github.io)

## About

This website showcases my academic profile, research publications, education, professional experience, and news updates. It is built with Jekyll using a customized [academicpages](https://github.com/academicpages/academicpages.github.io) theme.

## Sections

- **Home** - Brief introduction and research interests
- **News** - Latest academic and professional updates
- **Experience** - Professional and teaching positions
- **Education** - Academic qualifications (PhD, M.Sc., B.Sc.)
- **Publications** - Research papers and conference proceedings (powered by BibBase)
- **Speeches, Honors & Awards** - Recognitions, conference talks, and leadership roles
- **CV** - Downloadable resume
- **Contact** - Email, LinkedIn, and location

## Tech Stack

- [Jekyll](https://jekyllrb.com/) - Static site generator
- [GitHub Pages](https://pages.github.com/) - Hosting
- [BibBase](https://bibbase.org/) - Publication listing
- Theme based on [academicpages](https://github.com/academicpages/academicpages.github.io) / [Minimal Mistakes](https://github.com/mmistakes/minimal-mistakes)

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000`.
